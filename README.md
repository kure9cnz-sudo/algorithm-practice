# Algorithm Journey IDEA Practice Project

This folder is generated from your local `algorithm-journey-main` download.

## Modules

- `original`: selected source files copied as-is. Treat this as the answer/reference library.
- `practice`: same selected files, but most core implementation method bodies are replaced with `UnsupportedOperationException`. Comments, method signatures, main methods, random generators, comparators/right answers, and simple validators are kept as much as possible.

## How to Use

1. Open this folder in IntelliJ IDEA: `algorithm-journey-idea`.
2. Use the `practice` module for daily training.
3. When a method throws `TODO: implement ...`, implement it yourself.
4. Compare with the same file under `original` only after you have tried seriously.
5. Prefer files listed in `SELECTION.md`; they were chosen for Tokyo Science / old Titech C-group relevance.

## Exam-Focused Rule

Do not chase every file. For the C group, prioritize:

1. sorting and complexity
2. binary search
3. linked list, stack, queue
4. tree and heap
5. hash table and comparator
6. BFS/DFS and graph basics
7. grid/string DP: LCS, edit distance, minimum path sum, knapsack

Advanced topics above roughly LeetCode 2000+ were intentionally not copied into this project.
